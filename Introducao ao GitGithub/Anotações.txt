O Git Bash é um Terminal extendido para otimizar o uso do Git.

Comandos Inicias GIT/GITHUB
 
- git status - verifica se existe algum arquivo esperando para subir ao
repositório.
 
- git add . - adiciona o arquivo modificado, mas ainda não o sobe ao 
repositório.
 
- git commit - m " " - Inclui o arquivo modificado a area de atualização 
do git, mas ainda não o envia ao repositório.
 
- git push origin main - comando que sobe para a nuvem o arquivo atualizado.
